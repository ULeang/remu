# remu
a toy RISC-V emulator written in c(RV64IM)

## Features
- incomplete functions
- poor performance
- difficult to use
- terrible coding style
- and much more

## Installation

### dependencies
Arch
```
$ sudo pacman -S gcc riscv64-linux-gnu-gcc readline
```

### building
```
$ git clone https://github.com/ULeang/remu
$ cd remu && mkdir -p build
$ make
```
